### Prerequisites

- Node.js (v14.0.0 or higher)
- npm (v6.0.0 or higher)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/aryan2729/Time-Loop.git
   ```

2. **Navigate to project directory**
   ```bash
   cd Time-Loop
   ```

   ```bash
   npm install
   ```

   ```bash
   npm run dev
   ```
